X, Y Start

Determines hero start position

Hero will be set to that position
automatically after level loading.

"Start": { "X": 5, "Y": 2 }

Is time chamber?

Determines level as time chamber

  • true - no enemies at the level,
    doom countdown starts.
  • false - usual level.

"IsTimeChamber": false

.iI Danger

Determines random encounters frequency

  • IsZero - is 'X' and 'Y' equals 0?
  • X - bottom random chance border.
  • Y - top random chance border.

When you are walking on the map hero
"ToBattle" steps slowly decrement to
0. When it happens - battle begins.

After end of the battle "ToBattle" steps
will be recalculated and set as a random
value between 'X' and 'Y'.

Values must be less or equal 2 ^ 31.

Values can't be negative.

"Danger": { "IsZero": false, "X": 9, "Y": 27 }

If "IsTimeChamber" parameter is set
to true determines time to escape.
  • X - minutes.
  • Y - seconds.
! Stage foes

Determines foes on current location

"StageFoes": [0, 1, 2, 3]

$ Equipment

Determines treasure positions

  • "4:3" - short treasure position
    aka "X":4, "Y":3

"Equipment": { "4:3": { "Kind": 0, "Set": 1 }

K# Gates

Determines keys-gate positions

  • "3:2" - short key position
    aka "X":3, "Y":2
  • X - gate row.
  • Y - gate column.

"Gates": { "4:3": { "X":3, "Y": 2 } }

T Warps

Determines warp positions

  • "5:1" - short warp position
    aka "X":5, "Y":1
  • X - warp destination row.
  • Y - warp destination column.

"Warps": { "5:1": { "X":1, "Y": 1 } }

! Bosses

Determines boss positions

  • "2:2" - short boss position
    aka "X":2, "Y":2

"Bosses": { "2:2": 0 }

remember that it should not be
too short and easily passable.

long for the player to find all the
secrets and interesting places.
