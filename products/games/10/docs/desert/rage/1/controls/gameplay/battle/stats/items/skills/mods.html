How to determine skill power?

  • 0 - const value
  • 3 - SC * Hero SPL
  • 4 - SC * Hero (ATK + SPL)

Cure example

"Attributes": { "Dependency": 0, "Power": 85, "Value": 0 }

  • Power = const value = 85

  • Power = SC = 85
