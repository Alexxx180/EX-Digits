HP Health points
  • IsMax - is current HP matches max?
  • IsEmpty - is current HP equals 0?
  • IsSealed - is max HP equals minimum?

Values must be less or equal 65355.

Values can't be negative or float.

"Hp":{ "IsMax":true, "IsEmpty":false, "IsSealed":false,
"Minimum":0, "Current":300, "Max":500 }

EXP Experience

Determines experience at the start

  • IsMax - is current EXP matches max?
  • IsEmpty - is current EXP equals 0?
  • IsSealed - is max EXP equals minimum?

Values must be less or equal 65355.

Values can't be negative or float.

"Experience": { "IsMax": false, "IsEmpty": true, "IsSealed": false,
"Minimum": 0, "Current": 0, "Max": 10 }

+ Next level

Determines experience to the next levels

Values must be less or equal 65355.

Values can't be negative or float.

You can edit next stats in Next.json file.

{ "Minimum": 0, "Current": 0, "Max": 24 }

You need to duplicate this up to
the number of levels in game.

To note: make sure that your levels
count is less or equal 255 otherwise
hero will return to first level or
it may crash the game
