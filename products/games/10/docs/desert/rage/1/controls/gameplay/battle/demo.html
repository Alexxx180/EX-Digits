Find options in the right bottom corner of
battle area, before your standing hero.

Haven't any options shown?

Watch after progress in the Time bar.

When bar is filled options should shown.

Now you can select one from two routes
to continue.

Fight them all

You can make it in a various ways.

  1. Select Fight option.

    It's recommended to use this option
    against physically weaker monsters.

  2. Select Skills option.

    Search for fight skills like Torch.

    Select one. Some skills also need
    target enemy specified.

Wait for your next turn and repeat.

Run away

Simply select Escape option.

Not worked? Try route above.
